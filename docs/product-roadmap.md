# Product Roadmap

## MVP

- Local-only n8n JSON upload.
- Parser for workflow name, nodes, node types, trigger nodes, webhook nodes, HTTP request nodes, AI nodes, credentials references, connections, and documentation nodes.
- Heuristic scoring from 0 to 100 across reliability, security, maintainability, AI readiness, production readiness, and overall score.
- Structured report with critical issues, warnings, recommended fixes, production checklist, and next steps.
- Sample workflows for manual testing.

## Next

- Add Playwright coverage for the upload-to-report flow.
- Add report export to PDF.
- Add richer n8n graph analysis for upstream and downstream node relationships.
- Add opt-in saved reports after privacy, retention, and account boundaries are designed.
- Add industry-specific check packs.

## Later

- GitHub import for workflow JSON stored in repos.
- n8n Cloud import with user-authorized OAuth.
- Scheduled re-audits and drift detection.
- Team workspaces, reviewer comments, and organization policy templates.
